# 🌱 EcoTrack – AI Carbon Footprint Tracker

## 📌 Overview
EcoTrack is a cloud-integrated web application designed to estimate, analyze, and reduce individual and community-level carbon footprints.

The system converts user lifestyle activities into measurable carbon emissions and provides AI-driven insights to promote sustainable living.

---

## 🚀 Features
- 🌍 Carbon footprint calculation (kg CO₂/month)
- 🤖 AI-powered sustainability assistant
- 📊 Interactive dashboard with emission breakdown
- 🏆 Community ranking system (pincode-based)
- 🎯 Gamified eco-challenges
- 📈 Monthly trend analysis

---

## 🧠 System Approach
The application collects user data related to:
- Transportation
- Energy usage
- Dietary habits
- Lifestyle activities

These inputs are processed using standardized emission factors to estimate carbon emissions.  
An AI assistant provides personalized recommendations for reducing environmental impact.

---

## 🛠️ Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Supabase
- AI Integration: AI-based assistant (for sustainability recommendations)
